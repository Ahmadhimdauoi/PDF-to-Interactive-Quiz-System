# PDF to Interactive Quiz System

نظام ويب يحول ملفات PDF إلى اختبارات تفاعلية تعليمية

## الوصف
هذا النظام يتيح للمشرفين إنشاء اختبارات تفاعلية من ملفات PDF، مع إمكانية اختيار نوع الأسئلة واللغة. يوفر للطلاب واجهة سهلة لحل هذه الاختبارات والحصول على شرح فوري للإجابات.

## المميزات

### لوحة المشرف:
- رفع ملفات PDF للمقررات الدراسية
- تحديد عدد الأسئلة المطلوب
- اختيار نوع الأسئلة (صح/خطأ، اختيار من متعدد)
- اختيار لغة الواجهة
- رفع ملفات إضافية لتحسين جودة الأسئلة

### واجهة الطالب:
- عرض قائمة المواد المتاحة
- حل الاختبارات التفاعلية
- الحصول على شرح فوري للإجابات
- إعادة الاختبار مع ترتيب عشوائي للأسئلة

## المتطلبات
- Python 3.8 أو أحدث
- Node.js 14.0 أو أحدث
- pip (مدير حزم Python)
- npm (مدير حزم Node.js)

## التثبيت
1. قم بتثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

2. قم بتثبيت مكتبات الواجهة الأمامية:
```bash
cd frontend
npm install
```

3. قم بتشغيل التطبيق:
```bash
# في نافذة تيرمinal جديدة
python app.py

# في نافذة تيرمinal أخرى
cd frontend
npm start
```
